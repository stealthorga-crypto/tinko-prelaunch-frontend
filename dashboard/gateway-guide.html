How to Find Your Payment Gateway Details

Simple guide to fetching the credentials Tinko needs to recover your revenue.

Razorpay

What you need

API Key ID API Key Secret

Where to find it

  1. Go to Dashboard → Settings → API Keys
  2. Click Generate Live Key
  3. Copy Key ID and Key Secret

Stripe

What you need

Secret Key Webhook Signing Secret

Where to find it

  1. Go to Dashboard → Developers → API Keys
  2. Copy Secret Key (SK Live)
  3. For Webhook: Developers → Webhooks → Reveal Signing Secret

Cashfree

What you need

App ID Secret Key

Where to find it

  1. Log in to Cashfree Dashboard
  2. Go to Developers → Credentials
  3. Copy App ID and Secret Key

PayU

What you need

Merchant Key Merchant Salt

Where to find it

  1. Go to Dashboard → Profile → Integration Details
  2. Copy Merchant Key + Salt

PhonePe

What you need

Merchant ID Salt Key Salt Index

Where to find it

  1. Go to PhonePe PG dashboard → API Keys Section
  2. (Or check your email from PhonePe activation)

Paytm

What you need

Merchant ID (MID) Merchant Key

Where to find it

  1. Go to Dashboard → Developer Settings
  2. Copy MID and Merchant Key

CCAvenue

What you need

Merchant ID Access Code Working Key

Where to find it

  1. Go to Dashboard → Settings → API Keys
  2. Copy Access Code + Working Key

Juspay

What you need

API Key Merchant ID

Where to find it

  1. Go to Login → Developers Section
  2. Copy API Key
